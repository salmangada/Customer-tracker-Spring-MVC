# Spring MVC demo project.
## Guide
1. Download the project.
2. Create database `web_customer_tracker`
3. Update databse credential in `spring-mvc-crud-demo-servlet.xml` on line 35 and 36.
4. Run the script `customer.sql` (availabe in the project) in database.
5. Integrate the tomcat server in the workspace.
6. Run the application on server.
